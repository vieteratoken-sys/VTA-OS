# VTA Whitepaper (Lite)

## Introduction

Vietera Token (VTA) is a fixed-supply digital asset deployed on Solana
with a maximum supply of 5,000,000 tokens.

## Core Principles

- Scarcity
- Transparency
- On-chain verification
- Authority locked

## Supply Model

Max Supply: 5,000,000  
Inflation: None  
Mintable: No  

## Security

All token authorities have been revoked.
